## 2024-03-20 - Global Focus Ring Pattern
**Learning:** Adding a global `*:focus-visible` outline is a high-leverage way to immediately improve keyboard navigation across the entire application without needing to manually style focus states for every interactive component. It specifically targets keyboard users without affecting mouse user aesthetics.
**Action:** Consider implementing a global `:focus-visible` rule in the base stylesheet for projects lacking explicit focus styling for interactive elements.
